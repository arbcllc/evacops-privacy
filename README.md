# EvacOps Privacy Site

Static privacy-policy site intended for:

`https://arbcllc.github.io/evacops-privacy/`

## Review before publishing

1. Have ARBC business/legal counsel review the policy language.
2. Confirm `https://arbceng.com/` is the desired privacy-contact route.
3. Confirm the effective date in `index.html`.
4. Keep the policy aligned with the App Store privacy questionnaire and the app's actual data practices.

## Publish with GitHub Pages

This directory is published from the public `arbcllc/evacops-privacy` repository. Copy its contents to the repository root, then commit and push to `main`.

In GitHub:

1. Open **Settings > Pages**.
2. Under **Build and deployment**, choose **Deploy from a branch**.
3. Select the `main` branch and `/ (root)` folder.
4. Save and wait for the Pages deployment to complete.

The site contains no build step and no runtime secrets.
